Data Types

Javascript has 7 data types: String, Number, Boolean, Null, Undefined, Symbol and finally Objects with Arrays and Functions

Objects

An object is a data type in JS that is used to store a combination of data in a simple key-value pair.

Functions

A function is simply a bunch of code bundled in a section, that only runs when a function is called. It allow for organizing code into sections and code reusability.

Scope

The limits in which a variable exists.

  • Global scope
  • Functional scope
  • Lexical scope

Operators

Operators are reserved-words that perform action on values and variables.

  • Arithmetic: +,-,*,/,%,**
  • Assignment: =,+=,-=,*=

Coercion

When trying to compare different 'types', the JavaScript engine attempts to convert one type into another so it can compare the two values.

Conditional

It allows our program to run specific code only if certain conditions are met.
Ternary Operator:
(expression)? ifTrue: ifFalse;

Loop Statements

Loops are used to do something repeatedly. Lets say we have a list of 50 blog posts from DB and we want to print their titles. Instead of writing the code 50 times we would use loops.
